# 延迟优化

> 延迟是实时通信的关键指标，优化延迟对于良好的用户体验至关重要

## 📌 延迟来源

### 端到端延迟组成

```
┌─────────────────────────────────────────────────────────────┐
│                    端到端延迟组成                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────┐   ┌─────────┐   ┌─────────┐   ┌─────────┐    │
│  │  采集   │──►│  编码   │──►│  打包   │──►│  发送   │    │
│  │ 10-30ms │   │ 5-50ms  │   │  1-5ms  │   │  1-5ms  │    │
│  └─────────┘   └─────────┘   └─────────┘   └─────────┘    │
│                                                             │
│                     网络传输                                │
│                   10-200ms                                  │
│                                                             │
│  ┌─────────┐   ┌─────────┐   ┌─────────┐   ┌─────────┐    │
│  │  接收   │──►│  抖动   │──►│  解码   │──►│  渲染   │    │
│  │  1-5ms  │   │ 缓冲    │   │ 5-30ms  │   │ 5-20ms  │    │
│  └─────────┘   │ 20-100ms│   └─────────┘   └─────────┘    │
│                └─────────┘                                  │
│                                                             │
│  总延迟: 50-450ms (典型值: 100-200ms)                       │
└─────────────────────────────────────────────────────────────┘
```

### 延迟分类

| 类别 | 来源 | 典型值 | 优化难度 |
|------|------|--------|----------|
| **采集延迟** | 摄像头/麦克风 | 10-30ms | 中 |
| **编码延迟** | 视频编码器 | 5-50ms | 高 |
| **打包延迟** | RTP 打包 | 1-5ms | 低 |
| **网络延迟** | 传输+排队 | 10-200ms | 高 |
| **抖动缓冲** | 抗抖动 | 20-100ms | 中 |
| **解码延迟** | 视频解码器 | 5-30ms | 中 |
| **渲染延迟** | 显示输出 | 5-20ms | 中 |

### 3. 网络优化

```javascript
// 网络层优化
class NetworkOptimizer {
  constructor(pc) {
    this.pc = pc;
  }

  // 优先使用低延迟路径
  async preferLowLatencyPath() {
    const stats = await this.pc.getStats();
    const candidatePairs = [];

    stats.forEach(report => {
      if (report.type === 'candidate-pair' && report.state === 'succeeded') {
        candidatePairs.push({
          localCandidateType: report.localCandidateType,
          remoteCandidateType: report.remoteCandidateType,
          rtt: report.currentRoundTripTime,
          priority: report.priority
        });
      }
    });

    // 排序选择最低 RTT
    candidatePairs.sort((a, b) => a.rtt - b.rtt);

    return candidatePairs[0];
  }

  // 检测并避免中继
  avoidRelayIfPossible() {
    // 配置 ICE 优先级
    const config = {
      iceServers: [...],
      iceCandidatePoolSize: 10,
      // 优先直连
      iceTransportPolicy: 'all'
    };
  }
}
```

### 4. 抖动缓冲优化

```javascript
// 自适应抖动缓冲
class LowLatencyJitterBuffer {
  constructor() {
    this.minDelay = 10;      // 最小延迟
    this.maxDelay = 100;     // 最大延迟
    this.targetDelay = 30;   // 目标延迟
  }

  // 动态调整
  adapt(jitter, lossRate) {
    // 低丢包时可以更激进
    if (lossRate < 0.01) {
      this.targetDelay = Math.max(
        this.minDelay,
        jitter * 2
      );
    } else {
      // 高丢包需要更大缓冲
      this.targetDelay = Math.min(
        this.maxDelay,
        jitter * 4
      );
    }

    return this.targetDelay;
  }
}
```

### 5. 解码优化

```javascript
// 解码器优化
const decoderOptimizations = {
  // 硬件加速
  hardwareAcceleration: true,

  // 异步解码
  async: true,

  // 跳帧策略
  skipFrames: {
    // 当缓冲超过阈值时跳过非关键帧
    threshold: 3,
    skipNonKeyFrames: true
  }
};

// 实现跳帧逻辑
class FrameSkipController {
  constructor() {
    this.bufferedFrames = 0;
    this.skipThreshold = 3;
  }

  shouldSkipFrame(isKeyFrame) {
    if (this.bufferedFrames > this.skipThreshold) {
      // 缓冲过多，跳过非关键帧
      return !isKeyFrame;
    }
    return false;
  }

  onFrameDecoded() {
    this.bufferedFrames--;
    if (this.bufferedFrames < 0) this.bufferedFrames = 0;
  }

  onFrameReceived() {
    this.bufferedFrames++;
  }
}
```

## 📊 延迟监控

### 测量端到端延迟

```javascript
class LatencyMonitor {
  constructor(pc) {
    this.pc = pc;
    this.samples = [];
  }

  async measure() {
    const stats = await this.pc.getStats();
    let latency = {
      rtt: 0,
      jitterBufferDelay: 0,
      encodeDelay: 0,
      decodeDelay: 0
    };

    stats.forEach(report => {
      // RTT
      if (report.type === 'candidate-pair' && report.state === 'succeeded') {
        latency.rtt = report.currentRoundTripTime * 1000; // ms
      }

      // 抖动缓冲延迟
      if (report.type === 'inbound-rtp') {
        if (report.jitterBufferDelay && report.jitterBufferEmittedCount) {
          latency.jitterBufferDelay =
            (report.jitterBufferDelay / report.jitterBufferEmittedCount) * 1000;
        }
      }

      // 编码延迟（需要 remote-outbound-rtp）
      if (report.type === 'remote-outbound-rtp') {
        // 估算
      }
    });

    // 计算总延迟估算
    const totalEstimate =
      latency.rtt / 2 +          // 单向网络延迟
      latency.jitterBufferDelay + // 抖动缓冲
      30 +                        // 估算编码延迟
      20;                         // 估算解码+渲染延迟

    this.samples.push({
      ...latency,
      totalEstimate,
      timestamp: Date.now()
    });

    return latency;
  }

  getStats() {
    if (this.samples.length === 0) return null;

    const totals = this.samples.map(s => s.totalEstimate);
    return {
      current: totals[totals.length - 1],
      average: totals.reduce((a, b) => a + b, 0) / totals.length,
      min: Math.min(...totals),
      max: Math.max(...totals)
    };
  }
}
```

### 延迟告警

```javascript
class LatencyAlert {
  constructor(threshold = 300) {
    this.threshold = threshold;
    this.callbacks = [];
  }

  check(latency) {
    if (latency.totalEstimate > this.threshold) {
      const alert = {
        type: 'high-latency',
        latency: latency.totalEstimate,
        threshold: this.threshold,
        timestamp: Date.now()
      };

      this.callbacks.forEach(cb => cb(alert));
      return alert;
    }
    return null;
  }

  onAlert(callback) {
    this.callbacks.push(callback);
  }
}
```

```javascript
// 超低延迟配置（适用于游戏/远程控制）
class UltraLowLatencyMode {
  constructor(pc) {
    this.pc = pc;
  }

  async enable() {
    // 1. 最小抖动缓冲
    // (WebRTC 自动管理，通过统计反馈)

    // 2. 最低分辨率
    const sender = this.pc.getSenders().find(s => s.track?.kind === 'video');
    if (sender) {
      await sender.track.applyConstraints({
        width: 320,
        height: 240,
        frameRate: 30
      });

      // 3. 最低码率
      const params = sender.getParameters();
      params.encodings[0].maxBitrate = 200000;
      params.encodings[0].scaleResolutionDownBy = 1;
      await sender.setParameters(params);
    }

    // 4. 禁用 FEC（减少开销）
    // 5. 减少重传
  }

  getConfig() {
    return {
      resolution: '320x240',
      frameRate: 30,
      bitrate: 200000,
      fecEnabled: false,
      nackEnabled: true,
      jitterBufferMin: 10
    };
  }
}
```
