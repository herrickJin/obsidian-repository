## 📁 源码结构

```

│   ├── browser/             # 主进程 API
│   │   ├── api/             # API 实现
│   │   └── init.js          # 初始化
│   ├── renderer/            # 渲染进程 API
│   │   ├── api/
│   │   └── init.js

│
├── shell/                    # C++ 组件

│
```

## 🔨 编译 Electron

### 1. 环境准备

```bash
# macOS
xcode-select --install

# 安装 depot_tools
git clone https://chromium.googlesource.com/chromium/tools/depot_tools.git
export PATH="$PATH:/path/to/depot_tools"

# 安装 Node.js
brew install node

# Windows
# 安装 Visual Studio 2022
# 安装 Windows SDK
# 安装 Python 3
```

### 2. 获取源码

```bash
# 创建目录
mkdir electron-gn && cd electron-gn

# 使用 gclient 获取源码
gclient config --name "src/electron" \
  --unmanaged https://github.com/electron/electron

# 获取代码 (耗时较长)
gclient sync --with_branch_heads --with_tags
```

### 3. 编译

```bash
cd src

# 设置编译目标
gn gen out/Testing --args="import(\"//electron/build/args/testing.gn\")"

# 编译 (使用所有 CPU 核心)
ninja -C out/Testing electron

# 运行
out/Testing/Electron.app/Contents/MacOS/Electron
```

### 4. 编译参数

```python
# args.gn 常用配置
import("//electron/build/args/testing.gn")

# 调试模式
is_debug = true

# 优化级别
is_component_build = true

# 目标平台
target_cpu = "x64"  # "x64" | "arm64"

# 功能开关
enable_nacl = false
```

### 2. IPC 通信实现

```cpp
// shell/common/api/electron_api_ipc.cc

// ipcRenderer.send
void Send(const std::string& channel,
          blink::CloneableMessage arguments) {
  // 发送到主进程
  render_frame_->Send(
      new ElectronRendererHostMsg_Message(
          routing_id, channel, arguments));
}

// ipcMain.handle
void Handle(const std::string& channel,
            blink::CloneableMessage arguments,
            InvokeCallback callback) {
  // 处理渲染进程请求
  int request_id = next_request_id_++;
  pending_requests_[request_id] = std::move(callback);

  // 发送请求到主进程
  render_frame_->Send(
      new ElectronRendererHostMsg_Invoke(
          routing_id, request_id, channel, arguments));
}
```

### 3. Node.js 集成

```cpp
// shell/common/node_bindings.cc

// 初始化 Node.js 环境
void NodeBindings::Initialize() {
  // 创建 Node.js isolate
  isolate_ = node::CreateIsolate(node_args, node_exec_args);

  // 设置事件循环
  uv_loop_ = node::GetCurrentEventLoop(isolate_);

  // 加载 Electron 模块
  LoadElectronModule();
}

// 运行 Node.js 事件循环
void NodeBindings::RunEventLoop() {
  uv_run(uv_loop_, UV_RUN_DEFAULT);
}
```

## 🤝 贡献指南

### 1. 贡献流程

```
1. Fork 仓库
   ↓
2. 创建分支
   git checkout -b feature/my-feature
   ↓
3. 编写代码
   - 遵循代码规范
   - 添加测试
   - 更新文档
   ↓
4. 提交 PR
   - 填写 PR 模板
   - 关联 Issue
   ↓
5. 代码审查
   - CI 通过
   - Reviewer 批准
   ↓
6. 合并
```

### 2. 代码规范

```javascript
// JavaScript 规范
// 使用 Standard 规范
npm run lint

// C++ 规范
// 遵循 Chromium 代码风格
// 使用 clang-format
git cl format
```

### 3. 测试

```bash
# 运行所有测试
npm run test

# 运行特定测试
npm run test -- --grep "BrowserWindow"

# 运行 C++ 测试
ninja -C out/Testing electron_unittests
out/Testing/electron_unittests
```

### 4. 文档贡献

```bash
# 文档位于 docs/ 目录
# 使用 Markdown 格式

# 本地预览文档
npm run docs
```
