## 📖 Chromium 是什么

### 定义

Chromium 是一个开源浏览器项目，由 Google 于 2008 年发起。它是 Google Chrome 浏览器的基础，也是众多其他浏览器和应用的内核。

```
┌─────────────────────────────────────────────────────────────┐
│                    Chromium 开源项目                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐        │
│  │   Chrome    │  │   Edge      │  │   Opera     │        │
│  │  (Google)   │  │ (Microsoft) │  │             │        │
│  └─────────────┘  └─────────────┘  └─────────────┘        │
│                                                             │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐        │
│  │  Electron   │  │    CEF      │  │  WebView    │        │
│  │   应用      │  │   嵌入      │  │  (Android)  │        │
│  └─────────────┘  └─────────────┘  └─────────────┘        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### Chromium vs Chrome

| 特性 | Chromium | Chrome |
|------|----------|--------|
| 开源 | ✅ 完全开源 | ❌ 包含闭源组件 |
| Logo | 蓝色 | 彩色 (红黄绿) |
| 自动更新 | ❌ 无 | ✅ 有 |
| 媒体编解码器 | 有限 | 完整 (H.264, AAC) |
| 崩溃报告 | ❌ 无 | ✅ 有 |
| 使用统计 | ❌ 无 | ✅ 有 |
| PDF 阅读器 | ❌ 无 | ✅ 内置 |
| Flash | ❌ 无 | 曾经内置 |

## 📜 浏览器引擎发展史

### 时间线

```
1998 ─── Gecko (Mozilla)
   │
2003 ─── WebKit (Apple, 从 KHTML 分支)
   │
2008 ─── Chromium / V8 (Google)
   │
2013 ─── Blink (Google, 从 WebKit 分支)
   │
2015 ─── EdgeHTML (Microsoft, 从 Trident 分支)
   │
2018 ─── Edge 转向 Chromium
   │
2023 ─── 现代浏览器格局稳定
```

### 主流渲染引擎

| 引擎 | 开发者 | 使用浏览器 | 特点 |
|------|--------|-----------|------|
| **Blink** | Google | Chrome, Edge, Opera, Brave | 市场份额最大 |
| **WebKit** | Apple | Safari | iOS 强制使用 |
| **Gecko** | Mozilla | Firefox | 独立发展 |
| **EdgeHTML** | Microsoft | 旧 Edge | 已废弃 |

## 🔀 Blink 与 WebKit 的分离

### 历史背景

```
2008: Google 使用 WebKit 作为渲染引擎
      │
      ├── 但 WebKit 主要由 Apple 主导
      ├── Google 与 Apple 发展方向分歧
      └── 多进程架构难以在 WebKit 中实现
      │
2013: Google 宣布 Blink 从 WebKit 分支
      │
      ├── 移除 WebKit 依赖代码
      ├── 重构 DOM 实现
      └── 更激进的架构演进
```

### Blink 的改进

```javascript
// Blink 相比 WebKit 的主要变化:

// 1. 移除 WebKit 的抽象层
// WebKit: Platform -> Port -> OS
// Blink:  直接对接各平台

// 2. 新的 DOM 实现
// - Oilpan 垃圾回收 (已移除，改用 BlinkGC)
// - 更高效的 DOM 操作

// 3. 更快的发布周期
// - 每 6 周一个稳定版本
// - Canary/Dev/Beta/Stable 四个通道
```

### 分支时间点

```
WebKit (Apple)              Blink (Google)
     │                            │
     │  2013.4.3 分支点            │
     ├────────────────────────────┤
     │                            │
     ▼                            ▼
  Safari                      Chrome
  iOS Safari                  Edge
                              Opera
                              Electron
```

## 🌍 Chromium 生态系统

### 浏览器产品

```
基于 Chromium 的浏览器
│
├── Google Chrome      - 最流行的浏览器
├── Microsoft Edge     - Windows 默认浏览器
├── Opera              - 挪威浏览器
├── Brave              - 注重隐私
├── Vivaldi            - 高度可定制
├── Arc                - 新一代浏览器
└── Yandex Browser     - 俄罗斯浏览器
```

### 嵌入式框架

```
Chromium 嵌入方案
│
├── Electron           - Node.js + Chromium
│   ├── VS Code
│   ├── Slack
│   ├── Discord
│   └── Figma
│
├── CEF (Chromium Embedded Framework)
│   ├── C++ 嵌入
│   ├── 跨平台支持
│   └── 游戏内嵌浏览器
│
├── WebView2           - Microsoft 提供
│   └── Windows 应用嵌入
│
└── Android WebView    - Android 系统
    └── 混合应用开发
```

## 🔧 Chromium 版本发布

### 发布通道

```
┌─────────────────────────────────────────────────────────────┐
│                    Chromium 发布通道                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Canary (金丝雀)                                             │
│  ├── 每日更新                                                │
│  ├── 最新功能，可能不稳定                                     │
│  └── 适合开发者测试                                          │
│                                                             │
│  Dev (开发版)                                                │
│  ├── 每周更新                                                │
│  ├── 相对稳定的新功能                                        │
│  └── 适合早期测试                                            │
│                                                             │
│  Beta (测试版)                                               │
│  ├── 每月更新                                                │
│  ├── 接近稳定的功能                                          │
│  └── 适合预发布测试                                          │
│                                                             │
│  Stable (稳定版)                                             │
│  ├── 每 4 周更新                                             │
│  ├── 完全测试的功能                                          │
│  └── 生产环境推荐                                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 版本号规则

```
版本号格式: 主版本.次版本.构建号.Patch

示例: 122.0.6261.57
      │   │ │     │
      │   │ │     └── Patch 版本 (安全修复)
      │   │ └──────── 构建号
      │   └────────── 次版本 (分支)
      └────────────── 主版本 (每 4 周递增)
```
