# 🏠 jinmo 的个人知识库

> 基于 PARA 方法构建的第二大脑

## 📂 目录结构

```
jinmo-obsidian/
├── 00-Inbox/          # 收集箱 - 快速捕获所有内容
├── 01-Projects/       # 项目 - 有明确目标和截止日期
├── 02-Areas/          # 领域 - 持续关注的责任区域
├── 03-Resources/      # 资源 - 有价值的参考资料
├── 04-Archives/       # 归档 - 已完成/不再活跃的内容
├── 05-Daily/          # 日记 - 每日记录与复盘
├── Templates/         # 模板 - 所有笔记模板
├── Automation/        # 自动化 - 脚本和配置
└── Assets/            # 资产 - 图片、附件、PDF
    ├── images/
    ├── attachments/
    └── pdfs/
```

## 🎯 PARA 方法说明

### 00-Inbox 收集箱
- **用途**: 快速捕获所有想法、灵感、待整理内容
- **规则**: 每周清空一次，内容必须移到其他文件夹
- **入口**: QuickAdd 快捷键快速捕获

### 01-Projects 项目
- **定义**: 有明确目标、截止日期的任务
- **示例**: 
  - 开发某个功能
  - 准备面试
  - 学习某个技术栈
- **命名**: `Project-项目名-开始日期`

### 02-Areas 领域
- **定义**: 持续关注、需要维护的责任区域
- **示例**:
  - 健康
  - 财务
  - 职业发展
  - 人际关系
- **特点**: 没有明确终点

### 03-Resources 资源
- **定义**: 感兴趣的主题、有价值的参考材料
- **示例**:
  - 技术笔记（前端/Rust/音视频）
  - 读书笔记
  - 收集的文章
- **组织**: 按主题分类

### 04-Archives 归档
- **用途**: 存放已完成/不再活跃的内容
- **来源**: 其他文件夹归档而来
- **保留**: 保持原有结构便于查找

---

## 🚀 快速开始

1. 按 `Cmd+P` 打开命令面板
2. 输入 `QuickAdd` 选择模板
3. 开始记录

## ⌨️ 快捷键

| 功能 | 快捷键 |
|------|--------|
| 快速捕获 | Cmd+Shift+I |
| 新建学习笔记 | Cmd+Shift+N |
| 每日记录 | Cmd+Shift+D |
| 打开索引 | Cmd+Shift+K |
