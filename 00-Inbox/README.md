# 📥 Inbox 收集箱

> 所有内容的第一站

## 用途

- 💡 灵光一现的想法
- 🔗 待阅读的链接
- 📝 临时的笔记
- 🎯 待办事项
- 📎 临时附件

## 规则

1. **快速捕获**: 不要思考分类，先记录下来
2. **定期整理**: 每周至少清空一次
3. **不要囤积**: 内容必须移到其他文件夹

## 整理流向

```
Inbox → 判断 → Projects (有目标有期限)
              → Areas (持续关注)
              → Resources (参考资料)
              → Archives (不再需要)
              → 删除 (无价值)
```
